# Plugins

Stores optional future feature modules for Progress Claw OS.

## Planned Plugins

- `smart-grab/`: smarter claw movement and grab-assist behavior
- `lucky-mode/`: configurable lucky-mode gameplay behavior
- `tournament-mode/`: tournament gameplay structure
- `coin-system/`: coin, credit, and payment integration

Plugins are placeholders for future phases. No plugin runtime logic is implemented yet.
